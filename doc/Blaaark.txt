Blaaark, a color scheme with a unique touch.

==============================================================================
Color scheme options					Blaaark-options*

						*g:Blaaark_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:Blaaark_transp_bg = 0
<
